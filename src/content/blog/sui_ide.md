# Sui Move Analyzer 教程

## 安装

前提，安装 Rust

1. 在终端运行命令

`cargo install --git https://github.com/movebit/move --branch move-analyzer 2 move-analyzer`

2. 在 VS Code 中安装插件

[sui-move-analyzer](https://marketplace.visualstudio.com/items?itemName=MoveBit.sui-move-analyzer)
